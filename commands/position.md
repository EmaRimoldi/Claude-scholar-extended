# Position Command

Articulate your contribution against the field.

## Goal

Activates the `contribution-positioning` skill to select closest works, build differentiation matrix, draft contribution statement, and anticipate reviewer objections.

## Usage

```bash
/position
```

## Workflow

1. Read novelty-assessment.md (if available) and analysis-report.md
2. Activate `contribution-positioning` skill
3. Write: contribution-positioning.md

## Integration

- **Primary skill**: `contribution-positioning`
- **Prerequisite**: `novelty-assessment` output (recommended), `results-analysis` output
- **Feeds into**: `story-construction`, `ml-paper-writing` (introduction and related work)
